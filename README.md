# Tasktopia

## Overview
Tasktopia is a productivity application. Users can create, track, and complete tasks.

## Building
To build and run this project, you need to ensure you have docker-compose installed on your system and then just run:

`docker-compose up --build`

Then:
- to run backend, open `http://localhost:8080` in your browser
- to run frontend (web app), open `http://localhost:5173` in your browser

## Features
### Task Management
- User authentication (registration, login)
- Create, edit, and delete tasks
- Task categories
- Task priorities
- Task filtering and search functionality

## Tech Stack
- **Backend:** Java
- **Database:** PostgreSQL
- **Frontend:** React
